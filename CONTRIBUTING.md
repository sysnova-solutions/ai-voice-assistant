# Contributing

Thanks for contributing to SysnovaAi.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Run `npm install`
4. Copy `.env.example` to `.env`
5. Run `npm run dev`

## Before Opening a Pull Request

- Keep branding and product naming consistent
- Do not commit `.env` or API secrets
- Test the app locally
- Run `npm run build`

## Good First Contribution Areas

- UI polish
- Bug fixes
- Better docs
- Workflow integrations
- Branding and prompt cleanup
